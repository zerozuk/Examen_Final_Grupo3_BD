# Calculadora Científica Básica
Desarrollo de examen final (Calculadora Científica Básica) para la carrera de big data Una calculadora sencilla pero funcional, escrita en Python, que permite realizar operaciones matemáticas básicas y científicas.
# Características
Operaciones básicas: suma, resta, multiplicación, división.
Operaciones avanzadas: potenciación y raíz cuadrada.
Manejo de errores: detecta división por cero y raíces de números negativos.
Interfaz sencilla con menú interactivo.
Uso del módulo math para operaciones científicas.
# Funcionalidades
Opción	Operación
1	Suma (a + b)
2	Resta (a - b)
3	Multiplicación (a * b)
4	División (a / b) — con control de división por cero
5	Potenciación (a^b) usando math.pow()
6	Raíz cuadrada (√a) — con control de números negativos
7	Salir del programa
# Notas
El programa maneja entradas de tipo flotante (float) para mayor precisión.
Incluye mensajes de error amigables (división por cero, raíz de número negativo).
Ideal como proyecto educativo para principiantes en python.
